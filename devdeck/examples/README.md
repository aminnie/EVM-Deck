# Examples Directory

This directory contains example code demonstrating how to use various features of the devdeck package.

## Directory Structure

- **controls/**: Examples for Stream Deck controls
  - `update_text_example.py` - How to update text on Stream Deck keys at runtime

- **midi/**: Examples for MIDI functionality
  - `midi_control_example.py` - How to configure and use MIDI controls

## Usage

These examples are reference code showing configuration patterns and usage examples. They can be used as templates for your own implementations.

### Controls Examples

#### Update Text Example
Shows different ways to update text displayed on TextControl keys after the application is running.

### MIDI Examples

#### MIDI Control Example
Shows how to configure MIDI controls in your `settings.yml` file, including:
- CC (Control Change) messages
- SysEx (System Exclusive) messages
- Port configuration
- Icon configuration
